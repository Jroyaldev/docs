## Core workspaces

Each workspace is purpose-built for a specific ministry responsibility:

- <a href="/workspaces/dashboard">Dashboard</a>: scan key metrics and quick alerts.
- <a href="/workspaces/people">People</a>: steward households, follow-ups, and staff invites.
- <a href="/workspaces/prayer">Prayer</a>: moderate the prayer wall and celebrate answered requests.
- <a href="/workspaces/sermons">Sermons</a>: shepherd the preaching pipeline from draft to delivery.
- <a href="/workspaces/calendar">Calendar</a>: map church-wide and personal rhythms in one view.
- <a href="/workspaces/tasks">Tasks</a>: coordinate ministry projects on a kanban board.
- <a href="/workspaces/chat">Chat</a>: keep cross-team communication rooted in ministry context.
- <a href="/workspaces/team">Team</a>: assign roles, titles, and mention handles for everyone.
